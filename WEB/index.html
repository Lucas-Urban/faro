Inteligência Artifical para te ajudar

O Faro Um é um projeto tecnológico que tem como objetivo auxiliar donos de cães perdidos a resgatá-los das ruas. Ele utiliza técnicas de visão computacional e a cooperação entre usuários para ajudar nesse momento difícil.

Projeto

Nós utilizamos redes neurais artificiais para localizar o seu melhor amigo e trazê-lo de volta para casa, junto daqueles que você ama.

Encontrar PET

A função "Encontrar meu PET" permite que o tutor forneça imagens e informações básicas do seu cão perdido. O sistema utiliza uma Rede Neural para buscar possíveis resultados na base de dados e notifica o tutor sobre novos registros de "PET Encontrado".

Pet Encontrado

A função "PET Encontrado" permite que o usuário forneça imagens e a localização atual de um cão perdido. O sistema notifica possíveis tutores que estão buscando seus pets, com base nas características físicas do animal.
